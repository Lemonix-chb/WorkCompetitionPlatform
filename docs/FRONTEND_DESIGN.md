# 信息管理与智能评价系统 - 前端设计规范

## 📋 设计概览

**设计理念**: 三种角色、三种风格、统一体验
**设计风格**: 现代简约、扁平化、数据驱动
**主色调**: 蓝色系（体现专业性和可信度）

---

## 🎨 设计风格对比

### 学生界面 - 活力蓝色系
**设计理念**: 年轻、活力、易用
**色彩方案**:
- 主色: #4384F7 (明亮蓝)
- 辅色: #E5EFFF (浅蓝背景)
- 强调色: #1E4E97 (深蓝)
- 成功色: #38B66A
- 警告色: #F5AE2F

**视觉特点**:
- 圆润的卡片设计 (border-radius: 16px)
- 大图标和清晰的文字
- 鼓励性的插图和徽章
- 进度条和成就系统

**核心功能**:
- 赛事浏览与报名
- 团队组建（邀请/申请）
- 作品提交与管理
- 评审结果查看

---

- 数据色: #38B66A (绿) / #F5AE2F (黄) / #EA4A4A (红)

**视觉特点**:

### 管理员界面 - 深蓝色系

## 🎯 核心交互设计

### 1. 团队组建流程（学生）

```
创建团队 → 邀请成员 → 成员接受 → 团队确认 → 提交报名
    ↓          ↓          ↓
    │      搜索学号    查看邀请详情
    │      发送邀请    接受/拒绝
    │      等待响应    加入团队
    ↓
  或申请加入已有团队 → 队长审核 → 加入成功
```

### 2. 作品提交流程（学生）

```
填写作品信息 → 上传文件 → 自动解压校验 → AI初审 → 提交成功
                   ↓
            格式检查
            大小检查
            类型检查
                   ↓
            显示校验结果
                   ↓
            通过/失败
```

### 3. 评审流程（评审员）

```
查看待评列表 → 选择作品 → 查看AI报告 → 下载作品 → 填写评分 → 提交
     ↓                                          ↓
  筛选赛道                                  滑动评分
  查看状态                                  填写评语
```

## 🔤 字体规范

**主字体**: Inter (Google Fonts)

**字号规范**:
- H1: 32px (页面标题)
- H2: 24px (区块标题)
- H3: 20px (卡片标题)
- Body: 14px (正文)
- Caption: 12px (辅助文字)

**字重**:
- Bold (700): 标题
- Medium (500): 强调
- Regular (400): 正文

## 🎭 动效规范

### 过渡时间
- 快速: 150ms (按钮hover)
- 正常: 300ms (展开收起)
- 慢速: 500ms (页面切换)

### 缓动函数
- 默认: `cubic-bezier(0.4, 0, 0.2, 1)`
- 弹性: `cubic-bezier(0.68, -0.55, 0.265, 1.55)`
